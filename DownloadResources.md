# Download Tutorial mods.

To prepare for the coming tutorials we will download some sample mods from the Enfusion Workbench documentation website. 

This link will take you to the Bohemia public cloud storage. This will let you download the necessary tutorial mods for most of the lessons in this course.

[Link to the Bohemia Interactive Github Repo (download here).](https://github.com/BohemiaInteractive/Arma-Reforger-Samples/tree/main)

![alt text](/images/image-2.png)

We want to click on the green button that says `Code` and `select Download Zip.`
![alt text](/images/image-3.png)

This will download a zip file to your `Downloads` folder.

![alt text](/images/image-4.png)

We want to select the zip file and select `extract all` it is located on the toolbar. In the image it is highlighted.

![alt text](/images/image-5.png)

A save window will appear. Navigate to your `Arma Reforger/addons` folder and extract it there. This is **not** your `Arma Reforger` Workbench folder. Extract the folder.

![alt text](/images/image-6.png)

The `Arma Reforger` folder should open once the extraction is complete. If not, navigate to your game folder. Then open the `addons` folder and either search `Arma-Reforger-Samples`  and you should see the folder. 

![alt text](/images/image-7.png)

Let’s open the Enfusion Workbench and scan for these mods so they are ready for use in future lessons.

![alt text](/images/image-8.png)

Make sure to scan the your game `addons` folder.

![alt text](/images/image-9.png)

You don’t need to enter the sample folder - you can just scan the `addons` folder. The Workbench will find the mods inside the folder.

These are the sample mods that we will be using during some of the lessons ahead.
